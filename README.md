# API Key Manager - 用户使用说明

## 简介

API Key Manager 是一款安全、易用的 Windows 桌面 API Key 管理工具，帮助您安全存储和管理各种 API 密钥。

## 系统要求

- Windows 10 或更高版本（64位）
- WebView2 运行时（Windows 11 自带，Windows 10 可能需要单独安装）

## 安装

1. 下载 `api-key-manager.zip` 并解压
2. 将解压后的文件夹放置在您喜欢的位置
3. 双击 `api-key-manager.exe` 启动应用

## 首次使用

首次启动时，应用会自动创建数据目录并生成加密密钥。

**重要提示**：
- 数据目录位置：`./data/`（应用目录下的 data 文件夹）
- 请定期备份 `data/` 目录中的所有文件
- 如果丢失 `master.key`，所有数据将无法恢复

## 功能说明

### 添加 API Key

1. 点击右上角「➕ 添加」按钮
2. 填写表单：
   - 网站名称（必填）：例如 "OpenAI"
   - API Key（必填）：您的 API 密钥
   - 标签（可选）：添加分类标签，如 "AI"、"MCP" 等
   - 备注（可选）：添加备注信息
3. 点击「保存」

### 查看/复制 API Key

1. 在列表中找到要查看的 Key
2. 点击眼睛图标查看完整 Key
3. 点击复制图标复制到剪贴板

### 搜索和筛选

1. 使用顶部搜索框输入关键字
2. 支持搜索网站名、标签、备注
3. 点击左侧标签进行快速筛选

### 导入/导出数据

#### 导出数据
1. 使用导出功能（由前端实现）
2. 复制 `data/` 目录中的所有文件：
   - `data.json` - 加密的数据文件
   - `master.key` - 加密密钥
   - `*.sha256` - 校验和文件

#### 导入数据
1. 准备导出的 `data.json` 和 `master.key` 文件
2. 使用导入功能选择文件
3. 系统会自动备份当前数据，导入失败会自动恢复

## 安全说明

- 所有 API Key 使用 AES-256-GCM 加密存储
- 每个文件都有 SHA-256 校验和，防止数据篡改
- 数据存储在本地，不会上传到任何服务器
- 请妥善保管 `master.key` 文件

## 故障排除

### 应用无法启动

1. 确认已安装 WebView2 运行时
2. 检查是否有杀毒软件拦截
3. 以管理员身份运行

### 数据丢失

1. 检查 `data/` 目录是否存在
2. 检查 `data.backup/` 目录是否有备份文件
3. 尝试从之前的备份恢复
